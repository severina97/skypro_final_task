Git

  • git init - команда для инициализации нового репозитория Git в текущей директории. Это означает, что вы создаете новую версию контроля для проекта.
  • git clone - используется для создания локальной копии удаленного репозитория. Это позволяет вам работать с кодом на вашем компьютере.
  • git add - добавляет изменения в индекс (staging area), подготавливая их для следующего коммита.
  • git commit - сохраняет изменения в репозитории с сообщением, описывающим сделанные изменения.
